# Landing Image Pack | Prompts listos para producir

Direccion visual acordada para MiCuota.online:

- Paleta: azul profundo + blanco + amarillo.
- Sensacion: confianza financiera, simpleza y claridad.
- Audiencia: profesionales independientes.
- Mensaje clave: mobile-first, cobro por link, QR y suscripcion.
- UI: dashboard limpio, sin look bancario pesado.

## Mapping recomendado (archivo -> seccion)

- `/assets/landing/hero-main.jpg` -> Hero principal
- `/assets/landing/dashboard-mockup.jpg` -> Beneficios (producto)
- `/assets/landing/how-it-works-flow.png` -> Como funciona (flujo 3 pasos)
- `/assets/landing/professionals-collage.jpg` -> Beneficios/casos de uso
- `/assets/landing/final-cta.jpg` -> CTA final
