# Glosario

## Billing

Gestion de obligaciones de cobro: cuotas, vencimientos, deuda y reglas de recurrencia.

## Checkout

Experiencia donde el alumno o paciente completa el pago.

## Conciliacion

Proceso de comparar lo esperado por MiCuota con lo informado por el proveedor de pago.

## Dunning

Secuencia de recupero de pagos fallidos o vencidos: recordatorios, reintentos y mensajes segmentados.

## External reference

Identificador generado por MiCuota y enviado al proveedor para conectar notificaciones con operaciones internas.

## Ledger

Registro auditable de eventos financieros y operativos asociados a pagos.

## Payin

Entrada de dinero desde alumno/paciente hacia el flujo de cobro.

## Payout

Salida o liquidacion de dinero hacia el profesional.

## PaymentOperation

Operacion concreta creada para cobrar algo mediante un proveedor.

## PaymentEvent

Evento inmutable o sem_inmutable que describe algo ocurrido durante el ciclo de vida de un cobro.

## Provider reference

Identificador devuelto por el proveedor. En Mercado Pago, MiCuota guarda el `id` de preference o preapproval.

## Tenant

Cuenta u organizacion dentro de MiCuota.
